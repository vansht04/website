## 🌌 About

A personal portfolio built around a **liquid glass aesthetic** — soft blur, depth, transparency, and motion.

It’s designed to feel less like a static portfolio and more like an interactive space where projects live.

Minimal, but not boring.

---

## ✨ Features

- 🫧 Liquid glass UI with layered depth & blur
- 🌊 Smooth scrolling experience
- 🎮 Interactive 3D elements
- 🎞 Motion-driven animations
- 📱 Fully responsive design
- 🧩 Clean component-based structure
- ⚡ Lightweight & performance focused

---

## 🛠 Tech Stack

| Layer | Tech |
|------|------|
| ⚛️ Frontend | React 19 |
| ⚡ Build Tool | Vite |
| 🟦 Language | TypeScript |
| 🎨 Styling | Tailwind CSS |
| 🎞 Animation | Motion + Lenis |
| 🌐 3D | Three.js + React Three Fiber |
| 🚀 Deploy | Vercel |

## ⚡ Run Locally

```bash
git clone https://github.com/vansht04/website.git
cd website
npm install
npm run dev
```

📍 Open: `http://localhost:3000`

---

## 📦 Build

```bash
npm run build
npm run preview
```
